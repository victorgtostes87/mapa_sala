Sobre o Mapa de Sala

Versão atual: {{ versao }}

O sistema organiza salas, reservas, instrumentos, horários acadêmicos, tarefas da recepção e comunicações internas da Policlínica de Psicologia UVV.

Objetivo principal: reduzir conflitos de horário, retrabalho e falta de visibilidade no uso das salas.

Termo simples de uso

Este sistema é uma ferramenta operacional. Ele não deve receber evolução, diagnóstico, queixa clínica, conteúdo de sessão ou informações sensíveis de pacientes.

  1. Registre apenas dados necessários para organizar sala, horário, aluno, reserva e uso operacional.
  2. Não registre informações clínicas sensíveis em observações.
  3. Ao terminar em computador compartilhado, saia do sistema.
  4. Em caso de dúvida, não registre o dado e procure a coordenação.
{% if topico == 'recepcao' %}

Como usar: recepção

  1. Abra o Painel para ver pendências do turno.
  2. Use Horários abertos para encontrar alunos que ainda podem receber paciente ou triagem.
  3. Use Reservas para aprovar ou recusar pedidos de sala e acompanhar instrumentos.
  4. Use Afazeres como lista compartilhada, no lugar de post-its no monitor.
  5. No mapa, cadastre somente informações operacionais.
Se a informação pertence ao atendimento clínico, ela deve ficar no sistema clínico oficial.
{% elif topico == 'coordenacao' %}

Como usar: coordenação

  1. Use o Painel da coordenação para acompanhar alunos sem paciente, triagens abertas e horários ociosos.
  2. Use Relatório semanal para uma visão resumida do uso das salas e supervisões.
  3. Use Usuários para cadastrar, editar, inativar ou excluir cadastros criados por engano.
  4. Use Saúde e manutenção para acessar logs, backup, exportação e importação.
Ações críticas ficam com coordenador: excluir usuário, excluir agendamento, importar Excel, baixar/restaurar backup e apagar logs.
{% elif topico == 'professor' %}

Como usar: professor

  1. Abra Minha supervisão para ver somente os alunos vinculados ao seu usuário.
  2. Confira quais alunos têm paciente marcado, triagem aberta e horário reservado sem paciente.
  3. Use Pedidos de vagas para solicitar novos pacientes ou triagens para um aluno.
  4. Acompanhe o status do pedido antes de enviar outro parecido.
{% elif topico == 'aluno' %}

Como usar: aluno

  1. Abra Meus agendamentos para ver seus horários, salas e reservas.
  2. A tela separa horários com paciente/triagem marcada dos horários reservados ainda sem paciente.
  3. Use Reservar sala ou Reservar instrumentos com pelo menos 24 horas de antecedência.
  4. Use Horário com Coordenação para marcar conversa quando houver disponibilidade.
{% else %}

Como restaurar backup

  1. Avise a coordenação e combine um horário sem usuários usando o sistema.
  2. Baixe ou escolha o backup correto em formato .db.
  3. Na tela de Backup, envie o arquivo e digite RESTAURAR.
  4. Depois, faça reload do Web App no PythonAnywhere.
  5. Confira usuários, mapa, reservas e logs principais.
Restaurar backup troca o estado do sistema. Faça isso só quando houver erro real, importação incorreta ou perda de dados.
{% endif %}
